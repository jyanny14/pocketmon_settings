# 프로젝트 규칙

## 작업 히스토리 기록 (필수)

이 프로젝트에서 수행하는 **모든 작업**은 `docs/history.md`에 기록한다.

- 작업을 마친 직후 해당 파일을 열어 내용을 추가한다.
- 날짜별(`## YYYY-MM-DD`) 섹션에 bullet로 작성한다. 해당 날짜 섹션이 없으면 새로 만든다.
- 기록할 내용: 무엇을 변경/추가/삭제했는지, 왜 그랬는지(필요 시), 관련 파일 경로.
- 단순 질문 응답이나 탐색만 한 경우는 기록하지 않아도 된다. **파일 생성/수정/삭제, 커맨드 실행으로 상태가 바뀐 경우**는 반드시 남긴다.
- 파일/폴더가 없다면 먼저 `docs/history.md`를 만들고 작성한다.
