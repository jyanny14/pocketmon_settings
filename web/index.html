사용 방법

  1. 1. 데이터 둘러보기

    포켓몬 · 도구 · 특성 · 기술 네 탭에서 한/영 이름으로 검색. 타입 · 분류 · 종족값 · Champions 재밸런스 플래그로 필터링·정렬 가능.

  2. 2. 파티 구성

    파티 빌더에서 슬롯을 클릭해 포켓몬·폼·특성·도구 선택. "상세 설정" 을 열면 4기술 · Stat Points 6스탯 · 성격까지 지정. 분석 패널에 종족값/실효 스탯 합계 · 타입 커버리지 · HOME 연동 게임 노출.

  3. 3. 공유 · 저장

    "URL 복사" 로 어디서든 같은 구성 재현. "로컬 저장" 으로 브라우저에 여러 파티 보관. 헤더 좌상단 토글로 한/영 즉시 전환.

  4. 4. AI 분석

    "AI 분석" 버튼 → 6개 템플릿 (약점 · 교체 · 기술 조합 · 남은 슬롯 · 상대 파티 · 자유 질문) 중 선택해 복사. URL fetch 를 못 하는 AI 는 Champions 데이터 파일을 함께 다운로드해 채팅에 첨부하고 프롬프트는 별도 복사.
