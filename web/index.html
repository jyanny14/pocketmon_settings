사용 방법

  1. 1

    후보를 빠르게 좁혀보기

    지금 메타에서 쓸 만한 후보를, 막연함 없이 바로 찾으세요.

    포켓몬·도구·특성·기술을 한/영 이름으로 검색하고, 타입·폼·분류·재밸런스 여부까지 한 번에 걸러봅니다. 넓은 후보군을 실전용 선택지로 빠르게 압축하는 시작점입니다.

    한·영 이름 검색 · 타입/폼/분류 필터 · Champions 재밸런스 확인

  2. 2

    아이디어를 실전 파티로 바꾸기

    생각만 하던 조합을, 바로 테스트할 수 있는 6슬롯 파티로 완성해보세요.

    슬롯을 채우며 포켓몬·폼·특성·도구를 고르고, 상세 설정에서 4기술·Stat Points·성격까지 맞춥니다. 분석 패널이 종족값/실효 스탯 합계와 타입 커버리지를 한눈에 보여줍니다.

    6슬롯 빌드 · 4기술 설정 · Stat Points · 실효 스탯/커버리지 확인

  3. 3

    버전별로 저장하고 바로 공유하기

    좋은 파티는 한 번 만들고 끝나지 않습니다.

    URL 복사로 친구·디스코드·커뮤니티에 같은 구성을 그대로 공유하고, 로컬 저장으로 메타별 실험 버전을 여러 개 남겨두세요. 같은 파티를 언제든 이어서 수정하고 비교할 수 있습니다.

    URL 재현 · 브라우저 로컬 저장 · 메타별 버전 관리

  4. 4

    AI에게 바로 물어볼 질문 만들기

    "무슨 질문부터 해야 하지?" 라는 막힘을 없애줍니다.

    약점·교체·기술 조합·남은 슬롯·상대 대응·자유 질문 중 원하는 분석 방향을 고르면, 내 파티에 맞는 프롬프트를 바로 복사할 수 있습니다. URL을 못 읽는 AI 는 데이터 파일을 함께 내려받아 첨부하면 됩니다.

    6개 질문 템플릿 · 프롬프트 즉시 복사 · 데이터 파일 다운로드
