AI에게 물어볼 질문 만들기

현재 파티를 Claude·ChatGPT 같은 AI 에게 바로 붙여넣을 수 있도록 만든 질문 템플릿 모음. URL fetch 가 가능한 AI 는 링크로, 그렇지 않으면 인라인 데이터 버전으로 복사하세요.

  • 약점이 막연할 때
  • 빈 슬롯이 안 떠오를 때
  • 상대 파티 대응이 급할 때
  • 더블 리드·시너지가 헷갈릴 때

파티는 URL 쿼리 ?p=… 로 전달됩니다. ← 파티 빌더로 돌아가 구성을 바꾸기

진행 단계

  1. ✓ 파티 구성
  2. ✓ 질문 페이지
  3. 3 질문 선택
  4. 4 복사 · 붙여넣기
배틀 모드

더블 모드는 리드 조합·스프레드 기술·파트너 시너지 관점으로 질문을 재작성하고, 더블 전용 카드 3개를 추가로 보여줍니다.

📦 AI 가 데이터 fetch 를 못 하는 경우

  1. 1. 아래 버튼으로 Champions 전체 데이터 파일을 다운로드
  2. 2. AI 채팅에 해당 파일을 첨부 (drag & drop 또는 파일 업로드)
  3. 3. 원하는 카드의 "프롬프트 + 데이터 복사" 로 프롬프트를 따로 복사
  4. 4. 챗에 붙여넣고 제출 — AI 는 첨부된 데이터를 Champions 의 진실의 소스로 사용합니다

짧은 사용 팁

  1. 파티 빌더에서 바로 넘어오면 URL 자동 입력 파티 빌더에서 "AI에게 물어보기"로 오면 파티 URL이 프롬프트에 자동 반영됩니다. 직접 여기에 왔다면 파티 빌더에서 구성 후 다시 오세요.
  2. 질문은 하나씩 나눠서 약점 분석과 멤버 교체를 한 번에 묻기보다 카드 1개 = 질문 1개가 AI 답변 품질이 더 높습니다.
  3. AI가 링크를 못 읽으면 데이터 파일 첨부 위쪽 "Champions 데이터 파일 다운로드"로 받은 JSON 파일을 AI 채팅에 첨부하면, 프롬프트 안의 URL fetch 폴백이 자동 작동합니다.

자주 묻는 질문

파티 URL 없이도 쓸 수 있나요?

네. 프롬프트 미리보기를 열고 {{PARTY_URL}} 부분을 직접 수정하거나, "프롬프트 복사" 후 AI 채팅에서 파티 정보를 붙여 설명해도 됩니다. 단, 파티를 미리 구성하면 AI가 훨씬 정확하게 분석합니다.

Claude와 ChatGPT 모두 같은 프롬프트를 써도 되나요?

대부분 가능합니다. 두 AI 모두 파일 첨부와 URL fetch를 지원합니다. 더 긴 분석(고급 버전)에서는 원하는 출력 형식을 구체적으로 적을수록 답변 품질이 올라갑니다.

복사 버튼이 작동하지 않으면 어떻게 하나요?

브라우저가 자동 복사를 차단한 경우 프롬프트 미리보기(▸ 미리보기)를 열고 텍스트를 직접 선택해 복사하세요. HTTPS가 아닌 환경에서는 Clipboard API가 동작하지 않습니다.

더블 모드는 언제 쓰나요?

더블 배틀로 플레이할 때 상단 "더블" 버튼을 누르면 리드 조합·스프레드·파트너 시너지 관점으로 프롬프트가 재작성되고, 더블 전용 카드 3개(리드·스프레드·시너지)가 추가로 표시됩니다.

AI가 Champions에 없는 포켓몬을 추천하면 어떻게 하나요?

프롬프트 안에 Champions 데이터 파일을 첨부하라는 폴백 문구가 포함되어 있습니다. 데이터 파일을 AI에 첨부하면 Champions에 없는 포켓몬을 추천하는 오류가 크게 줄어듭니다.
