# Policy
#
# 1. AI training crawlers — block everything. Our dataset derives from
#    serebii / PokeAPI; we don't want Nintendo IP baked into model weights.
# 2. User-triggered AI browsing (ChatGPT-User, ClaudeBot, etc.) — allow.
#    When a user pastes a URL into an AI tool the fetch should succeed.
# 3. Everything else (search engines, generic crawlers) — allow. Search
#    indexing is how real users discover the site; other Pokémon fan
#    references (Serebii, Bulbapedia, Smogon, PokemonDB) all index openly.
#    The share-only prompts.html stays out of indexes via its own
#    <meta name="robots" content="noindex"> tag.

Disallow: /
